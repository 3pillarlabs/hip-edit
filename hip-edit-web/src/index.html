HIP Edit

About

HIP Edit is a collaborative editor well suited for interviews. It is a recursive acronym that stands for HIP Interview Process Editor.

Hosting

Serverless application hosted on S3, with API gateway and AWS Lambda.

Tech Stack

  • Angular 5
  • Angular Material
  • Express JS
  • Troposphere
  • NodeJS
  • Python
  • Apache ActiveMQ

Others

  • Web fonts from Google
  • Material Icons
  • Jasmine
  • Karma
